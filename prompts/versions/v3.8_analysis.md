## 🤖 VLM一致性评估流程

### **评估维度**

```python
评估维度 = {
    "细节保持": {
        "吊坠形状":原图吊坠轮廓是否保持
        "纹理质感":金属纹理/宝石镶嵌是否一致
        "颜色":金属颜色/宝石颜色是否准确
        "刻字图案":原图刻字/图案是否保留
    },
    "防幻觉": {
        "链条幻觉":是否生成原图不存在的链条细节
        "装饰幻觉":是否添加原图不存在的装饰
        "材质幻觉":是否改变金属材质类型
    },
    "角度转换": {
        "正面朝向":是否正确转换为正面视角
        "吊坠朝向":吊坠是否正面朝向观众
    },
    "构图质量": {
        "吊坠占比":是否达到60-70%画面占比
        "特写效果":是否形成特写效果
    },
    "背景质量": {
        "纯白背景":背景是否纯白#FFFFFF
        "背景残留":是否有原图背景残留
    }
}
```

### **VLM评估提示词模板**

```markdown
# 一致性评估任务

## 输入
- 原始图片: [input_image]
- 生成图片: [output_image]

## 评估维度

### 1. 细节保持 (权重40%)
请对比原图和生成图的吊坠:
- 形状轮廓是否一致?
- 纹理质感是否一致?
- 颜色是否准确?
- 刻字/图案是否保留?

评分: 1-10 (10=完全一致)

### 2. 防幻觉 (权重40%)
请检查生成图:
- 是否生成原图不存在的链条细节?
- 是否添加原图不存在的装饰?
- 是否改变金属材质?

评分: 1-10 (10=完全无幻觉)

### 3. 角度转换 (权重10%)
- 是否正确转换为正面视角?
- 吊坠是否正面朝向?

评分: 1-10 (10=完全正确)

### 4. 构图 (权重5%)
- 吊坠是否占60-70%画面?
- 是否形成特写效果?

评分: 1-10 (10=完全符合)

### 5. 背景 (权重5%)
- 背景是否纯白?
- 是否有原图背景残留?

评分: 1-10 (10=完全纯净)

## 输出格式

总分 = Σ(维度得分 × 权重) / 10

请输出:
1. 各维度得分 (1-10)
2. 总分 (1-10)
3. 主要问题列表 (如果有)
```
